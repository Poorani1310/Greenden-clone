# Greenden-clone
Website for a nursery shop
This website is for a nursery shop Ivy Garden and it has a home page then a Products page which list all the pics of the plants available in the nursery and a contact for to enquire for the products available in the nursery.

The techologies used:
1. HTML
2. Tailwind CSS

The site was responsive for the mobile screen and the large laptop screen.
Used Tailwind CSS for the responsiveness of the site.
Used small animations for the products image to get used to the transform.

Use the link to view the site : https://poorani1310.github.io/Greenden-clone/
